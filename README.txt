Dependencies
============

Python 3 - https://www.python.org/

Requests library - http://docs.python-requests.org/en/master/

jQuery - https://jquery.com/

ZURB Foundation - http://foundation.zurb.com/


Voorbeelden
===========

$ python3 tabelpage.py

Genereert een tabel.html, gebruik makend van de argumenten in client.py.
Een eventueel al bestaande tabel.html wordt vervangen.


$ python3 objectpage.py <objectnummer>

Genereert een object.html.
Een eventueel al bestaande object.html wordt vervangen.


Aanvullende informatie
======================

De panelen met titel en details kunnen getoond of verborgen worden door op de
'buttons' bovenin de viewport te klikken.

De achtergrondafbeelding schaalt mee met de afmetingen van de viewport.
